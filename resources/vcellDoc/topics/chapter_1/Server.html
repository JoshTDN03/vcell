Server

The Server menu bar option allows the user to change the server settings for the current user of VCell.



Clicking the Server menu displays the following selections

  • Change User... - allows user to change the user of the currently open VCell document by providing a different user name and password.
  • Update Registration Info... - allows user to update his/her registration information.
  • Reconnect - allows user to reconnect to the server. The status bar displays the connecting status.
  • Change Server - is currently disabled in this menu.
  • Local - is currently disabled in this menu.

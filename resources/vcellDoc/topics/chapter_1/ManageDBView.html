Manage Database -> View

The Manage Database -> View menu option allows the user to change view selections in Database Manager.



The Manage Database -> View menu sequence has the following options:

  • Status Bar : allows the user to display/hide the status bar at the bottom of the document window.
  • Latest Version Only : displays only the latest version of all the models that are accessible to the user (user's own and those shared with the user or those that are public models.).

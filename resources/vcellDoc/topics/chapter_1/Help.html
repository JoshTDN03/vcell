Help

The Help menu bar option provides information of the currently launched VCell version.



Clicking the Help menu shows the possible menu item selections. Selecting

  • Online Help - launches a separate web browser that goes to the Virtual Cell Intro web page.
  • About The Virtual Cell - pops up a dialog that displays the Version and edition of the currently launched VCell.

  • Virtual Cell License Information - shows a dialog that displays the license information for the Virtual Cell.

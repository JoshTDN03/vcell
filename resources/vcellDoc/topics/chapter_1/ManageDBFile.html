Manage Database -> File

The Manage Database -> File menu option allows the user to open documents from the Database Manager. NOTE: this menu sequence requires that a model or one of its editions/versions (viewed by expanding the tree) be selected.



The Manage Database -> File menu sequence has the following options:

  • Open Latest Edition : This opens the latest edition/version of the selected model.
  • Open Selected Edition : This opens the selected edition/version of the model.
  • Export : exports the selected model/edition in the chosen format. See also File -> Export.
  • Exit : exits the Virtual Cell.

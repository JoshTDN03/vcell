Database Sources

The Virtual Cell provides access to a few database repositories, internal and external. The internal Virtual Cell database repository stores the models created in the Virtual Cell in its native format. In addition, access to external databases such as BioModels Database, Pathway Commons and Sabio-RK is provided.



The database sources in the Virtual Cell are accessed from the 'Database' panel on the lower left corner of any VCell document window. This panel has 3 tabs:

  • VCell Database
  • BioModels.net Database
  • Pathway Commons Database
  • Sabio-RK Database

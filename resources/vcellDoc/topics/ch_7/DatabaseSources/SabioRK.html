SABIO-RK - Biochemical Reaction Kinetics Database

SABIO-RK is a curated database that contains information about biochemical reactions, their kinetic rate equations with parameters and experimental conditions.



Search Sabio-RK web site for reactions by typing an associated term in the 'role' text box and pressing the 'Search' button to download a list of Sabio entities that match. Searches can be refined using the 'advanced' category drop-down selector.

Click on a row from the search list to load a list of the entity components that can be used in VCell. See Pathway Import for information on how to import selected components into VCell.

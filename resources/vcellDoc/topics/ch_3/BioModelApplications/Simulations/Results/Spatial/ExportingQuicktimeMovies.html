Exporting Quicktime Movies

Quick Time Movies: Select Range under the Time Interval for exporting a Quick Time movie, and, if applicable, whether you want a volumetric movie or slices or RIOs selected in the View Data tab. A dialogue window will appear in which you specify Movie Settings.

Once you select variable and time ranges and press Start Export you will see a Movie Settings dialog.



When QuickTime is selected for particle data, the server will create a movie using the VisIt visualization package.

  • Encoding Format: Select JPEG or GIF for the frames
  • File Compression: Choose anywhere between high compression (smaller file size) and lossless (larger file size)
  • Variables Display Composition: If you have multiple variables, you may select from two options. Separate makes a different movie for each variable, combined combines the results for all variables in a single movie.
  • QuickTime Format: Regular Quicktime (.mov) or QuickTime VR
  • Volume Variable Membrane Outline Thickness
  • Variables Mirroring: You can extend the image size for symmetric data sets by selecting the mirroring option.

  • Movie Duration: Choose how long you want your movie to be in seconds

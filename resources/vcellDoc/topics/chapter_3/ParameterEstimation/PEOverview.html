Parameter Estimation Overview

NOTE: Running parameter estimation is not available in the Virtual Cell release version. If you have old parameter estimation tasks saved, you may still be able to view the results under "Fitting"->"Parameter Estimation". New COPASI based parameter estimation is available in the Virtual Cell Beta version. You may get it from Virtual Cell Web site (http://vcell.org).

A detailed description of COPASI optimization methods is introduced. And followed by a section to guide user on how to do parameter estimation in the Virtual Cell Beta version.

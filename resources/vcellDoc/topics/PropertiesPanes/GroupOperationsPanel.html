Entities Management Toolbar

The buttons shown in this toolbar allow the user to perform various useful operations on selected entities within the Entities Diagram View and the Entities Table View .

This toolbar shows buttons allowing group, delete and link operations on selected entities. There is also present a Search feature which filters entities by string.

The user can perform various operations upon these entities, either on individually selected entities or multiple selected entities.

Linking. Single / multiple entities can be imported within a model. Following this process, equivalent entities to those pathway entities being imported will be created within the BioModel. A permanent link will preserve the relationship between the BioModel and the Pathway entities. To do this select one or more entities and press the Physiology Links button, then use the option Import into physiology...

The Import into physiology Dialog box is opened. Here the user is allowed to edit the values for the compartment and stoichiometric coefficient. The default values is c0 for the compartment and it may be sufficient for most cases. However, the user may take advantage of the location property of many pathway entities and manually edit the location field. Once that step is finished the imported entities are displayed within the Reaction Diagram View. In the context of this example c0 is extracellular region, c1 is the cytosol and Membrane0 is the plasma membrane.

Returning to the Pathway Diagram View it can be noticed that the linked entities display a small 'L' mark within their circle representation, indicating their linked status.

Existing groups can be grouped in even larger groups. Also groups can be broken down into their original subcomponents using the Ungroup option.

A particular type of grouping applies to reactions. The Collapse / Expand commands will hide / restore the entities associated with a selected reaction.

Deleting. Single / multiple entities can be deleted from the view using the Delete Selected button. Note that linked entities cannot be deleted as long as the link exists. In the case of selections where some entities can be deleted and some others can not, a warning dialog box will display pertinent information.

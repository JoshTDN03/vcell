Pathways Object Properties Pane

The object properties for any selected pathway object are shown in this pane.

In the first column is displayed the name of the property, in the second column is displayed the value of the property.

No properties can be edited. Blue items represent hyperlinks to various resources on the web, all the rest is represented in dark red.



Please check online documentation for a complete list of the properties associated to BioPAX classes.

Double click a property value depicted in blue and additional web resources related to that object / property will be displayed in a browser window.

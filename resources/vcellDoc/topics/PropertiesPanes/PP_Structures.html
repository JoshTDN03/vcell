Structure Properties Pane

The structure properties pane allows editing of the structure name and free text annotations. It is also shows size variable name and voltge variable name (for membranes).



At the top, there is the one-line structure name text field. Below, there is the one-line size variable name text field. For membranes, there is the one-line voltage variable name text field below that. Below, using most of the panels space, is the multi-line free text annotation text field.

Structure name and free text annotations can be changed by editing the appropriate text fields. A structure name will be rejected if already in use by another structure.

Structure Properties Pane

The structure properties pane allows editing of the structure name and free text annotations. It is also shows size variable name and voltge variable name (for membranes).

Compartment Structure Properties

  • structure name: Name of selected structure in the model.
  • size variable name: Variable having numerical value equal to selected compartment's size in a simulation geometry.
  • annotation: User notes associated with selected compartment structure.

Membrane Structure Properties

  • structure name: Name of selected structure in the model.
  • size variable name: Variable having numerical value equal to this compartment's size in a simulation geometry.
  • Voltage variable name: Variable having numerical value equal to (voltage(positive feature) - voltage(negative feature))
  • Positive (inside feature): Compartment (non-membrane) used in calculation of "voltage variable"
  • Negative (outside feature): Compartment (non-membrane) used in calculation of "voltage variable"
  • annotation: User notes associated with selected membrane structure.

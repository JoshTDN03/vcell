Properties Panes

The bottom right pane of the VCell document window contains an Object Properties tab that displays properties of any entity (e.g. species, reaction, structure) selected in the main View Pane of the document window.

Properties are specific to the item selected. The different properties displayed include:

  • Structures
  • Species Properties
  • Reactions
  • Membrane Flux
  • Stoichiometry
  • Pathway Information
  • Parameters

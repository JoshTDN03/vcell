Parameters Properties Pane

The parameter properties pane allows editing of the parameter attributes such as name, expression, units and free text annotations. Note:

  • Description cannot be changed for any type of parameter.
  • For 'Reaction' parameters that are not user-defined, only Name and Expression of parameter can be changed.
  • For 'Application' parameters, only their Expression can be changed.



Each of the parameter attributes are displayed in a text field in the properties pane. Depending on the type of parameter, the corresponding text fields are enabled or disabled.

Clicking inside an editable text field allows the user to change the value of the appropriate attribute for the parameter. If the user tries to change the name of a parameter by typing a name that already exists, an error dialog is shown, preventing the user from using the name.

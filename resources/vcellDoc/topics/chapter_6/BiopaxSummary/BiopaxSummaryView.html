Entities Detailed View

Displays the entities within the Entities Diagram View in text format.

The view has 2 sections. The Summary View is displayed first, followed by the Detailed View.

The Summary View shows the list of entities present in the Pathway Diagram View using the general format of Type, Name, ID.

The Detailed View shows, for each and every entity, the equivalent of the Entity Properties Pane in text format and with each inner sub-entity expanded.



All the information in this view is read-only, no user actions can be performed.

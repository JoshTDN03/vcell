Entities Table View

A table of pathway entities can be used to examine their properties and their link status. It is the table format representation of the Entities Diagram View.

The table view shows the pathway entities, their type and also shows which of these entities (if any) are linked to corresponding entities within the physiology.



It can be noticed that the selected reaction and its dependents are already linked to their respective physiology equivalents.

The bottom pane of the window displays a Entity Properties Pane. In the sample image presented below you can see the properties of the biochemical reaction selected in the Pathway Object View above.



More entities can be added from the Imported Pathway View.

The user can perform various operations upon these entities, either on individually selected entities or multiple selected entities using the Entities Management Toolbar.

Of particular interest is the process of importing pathway entities into the BioModel. Following this process, equivalent entities to those pathway entities being imported will be created within the Physiology.



The table items can be sorted alphabetically by any column by clicking on the column header. As an added capability, the items in the table can be filtered in real time by typing a string inside the Search field.

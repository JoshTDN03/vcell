Species

In BioModels, species are used to create variables (molecular species) in reactions and fluxes. Species can exist in both membranes and compartments (volumes) of the cellular structures. Species are unique pools of molecules within a single compartment. If the same molecular component is present in multiple compartments, the compartment name is appended to the species name to retain a unique identifier, e.g. species_compartment.

The Species Tab diplays a table of the species name and the structure that contains it. The Species Properties pane is shown in the bottom right pane

Edit A species name by highlighting and typing in either table or within the properties pane in the lower window.

Filter the list of species using the Search Box to enter a species name or a structure name.

Delete species by highlighting and using the "Delete Selected" button below the table.

The Pathway Link button is available to retrieve information from pathway databases. Add annotation about the species within the Species Properties pane in the bottom window.

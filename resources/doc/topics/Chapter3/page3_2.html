3.2 Fitting recovery curve
After loading the FRAP image and setting up all the required ROIs, users can proceed to fit recovery curve. Based on the bleaching type, Virtual Frap estimates the diffusion rate and mobile fraction that best fit the experimental data. We shall describe how to fit recovery curve below (Figure 3-4 shows the fitting recovery curve).

Fit recovery curve - When click on the “Fit Recovery Curve” tab, the fit recovery curve is automatically completed by using Circular Disk as default bleaching type.

Choose bleaching type - Three bleaching types are built in Virtual Frap, which are Circular Disk bleach, Gaussian Spot bleach and Half Cell bleach. Users just need to click on the radio button to select proper bleach type. Once the bleaching type has been changed, the curve fitting will be updated automatically.

View recovery curve - The time course of recovery equation and the experimental data will be displayed together in the plot panel. Users will see the plot view by default. Clicking on table button allows users to see the detailed data in table.

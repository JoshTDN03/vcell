3. Quick Tour of Virtual Frap
Virtual FRAP estimates and evaluates diffusion rates from FRAP experimental data in four steps. The four steps are:

  • Loading FRAP experiment data and defining regions of interest (ROI).

  • Primary estimation on a single diffusion rate by fitting a redistribution curve based on normalized temporal data under the bleached area.

  • Refined estimation on a primary diffusion rate and an optional secondary diffusion rate based on normalized temporal and spatial data.

  • Spatial visualization by constructing and fitting a spatial model for the FRAP data using VCell spatial solver.

3.1 Manipulating FRAP Experimental Data

Figure 3-1 illustrates the Virtual Frap user interface for loading and processing FRAP images. Virtual Frap image tool consists of several toolboxes to manipulating images. These are:

  1. Advanced ROI Toolbox - To allow importing region of interest, clearing ROI and plotting ROI time course.

  2. ROI Toolbox - To identify the specific ROI that is being defined.

  3. Drawing Toolbox - To enable painting, erasing and filling on the image.

  4. Adjustment Toolbox - To allow zooming in/out, increasing/reducing image contrast and cropping image by customized or default rectangular window.

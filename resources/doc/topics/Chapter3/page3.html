3. Quick Tour of Virtual Frap
Virtual FRAP estimates and evaluates diffusion rates from FRAP experimental data in four steps. The four steps are:

  • Loading FRAP experiment data and defining regions of interest (ROI).

  • Primary estimation of a single diffusion rate.

  • Refined estimation of a primary diffusion rate and an optional secondary diffusion rate.

  • Spatial visualization by constructing and fitting a spatial model for the FRAP data using VCell spatial solver.

3.1 Manipulating FRAP Experimental Data

Figure 3-1 illustrates the Virtual Frap user interface for loading and processing FRAP images. The menus of the Images interface only become active after an image has been loaded. The Virtual FRAP’s “Images” tab consists of several tools for manipulating images. These are:

  1. ROI Toolbox - To help automate defining ROIs and plotting ROI time course data.

  2. ROI Selection box - To identify the specific ROI that is being defined.

  3. Drawing Toolbox - To enable painting, erasing and filling on the image.

  4. Image Adjustment Toolbox - To allow zooming in/out, increasing/reducing image contrast and cropping image by customized or default rectangular window.

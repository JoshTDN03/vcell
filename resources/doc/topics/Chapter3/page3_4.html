3.4 2D Results
A spatial simulation is conducted after getting the initial parameters. The spatial simulation can be updated with new set of parameters in “Adjust Parameters” tab as aforementioned. The simulation results are used to compare with the original FRAP experimental. There are data viewers in the “2D Results” panel. The two viewers are,

  • The simulation results viewer - The simulation results show how the diffusion rate works in the cell. The immobile data, mobile data and combined data are shown in the “Variable” list. When users select one of the options, the corresponding data set will be loaded in the viewer.

  • The original data and ROI rings viewer - The original data are saved in two forms which are raw data and normalized raw data. The normalized raw data is the proper dataset to compare with the simulation results because the simulation runs based on the normalized data. The ROI rings are sample at different distances from the user defined initial bleached region. The ROI rings enable users to observe how diffusion works in others regions of cell besides the bleached area.

The spatial simulation is done when users click on the “Adjust Parameters” tab. When all the required data are ready, the simulation will complete automatically. (Figure 3-6 shows the spatial simulation results)

To understand and export the spatial simulation results. Please refer to Virtual Cell documentation at http://vcell.org/userdocs/Rel/user_guide.pdf for chapter 8 and chapter 9.

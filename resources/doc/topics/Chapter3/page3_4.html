3.4 2D Results
The spatial simulation is conducted initially by the parameters from “Initial Parameters” tab. It can be updated with new set of parameters in “Adjust Parameters” tab as aforementioned. The simulation results are used to compare with the original FRAP experimental data. The two sets of data are displayed in Simulation results viewer and Original Virtual Microscopy Data viewer.

  • Simulation results viewer – The simulation is conduced by VCell spatial solver. The immobile data, primary mobile data, secondary mobile data (if users have selected) and combined data are shown in the “Variable” list. The combined data is a sum of primary mobile data time primary mobile fraction, secondary mobile data time secondary mobile fraction (if users have selected) and immobile data time immobile mobile fraction. When users select one of the options, the corresponding data set will be loaded in the viewer.

  • The original data and ROI rings viewer – The original data are saved in two forms which are raw data and normalized raw data. The normalized raw data is the proper dataset to compare with the simulation results because the simulation runs based on the normalized data. The ROI rings sample at different distances from the user defined initial bleached region. Through ROI rings, users are able to observe how diffusion works in different regions of cell besides the bleached area.

There are two kinds of view in this panel when taking into consideration that different monitors have different resolutions. Parallel view requires higher resolution, which allows users to compare the simulation results and original data side by side. Tabbed view allows users to see one data set each time.

The spatial simulation is performed when users click on the “Adjust Parameters” tab. As long as all the required data are ready, the simulation will complete automatically. (Figure 3-6 shows the spatial simulation results)

To understand and export the spatial simulation results. Please refer to Virtual Cell documentation at http://vcell.org/userdocs/Rel/user_guide.pdf for chapter 8 and chapter 9.

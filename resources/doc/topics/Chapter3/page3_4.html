3.4    Selecting FRAP Model
After loading the FRAP image and setting up all the required ROIs, users can proceed to select possible models that may fit the data. One possible model is diffusion with one diffusing component and another is diffusion with two diffusing components. Users can choose either one or both of them.  The “Choose Model Type” wizard has only one page. To select the model users just need to check the checkbox related to the specific model type.

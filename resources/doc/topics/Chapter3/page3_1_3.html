3.1.3 Defining ROIs on FRAP Experimental Data
By using ROI Toolbox and Drawing Toolbox, users are able to manually draw the ROIs on FRAP images. Users are compulsory to define whole cell area, initial bleach area and representative background area. Without defining all the ROIs, the further procedures will not be available to users.

Specify a ROI - The ROI Toolbox is on top of the image. Click on the radio button to specify the ROI that you are going to define.

Draw ROI area - Once the paint button is pressed, the paint mode will be on until another button is selected. When drag the mouse over the image at the paint mode, the area that mouse has passed will be highlighted and saved as specified ROI.

Erase ROI area - Select the erase mode by clicking on erase button . Dragging the mouse over the painted area will erase the part that the mouse has passed.

Fill ROI area - Select the fill mode by clicking on erase button . If users click the mouse again in a closed area, the area will be automatically filled.

Virtual Frap has a couple of advanced ROI tools to speed up the defining of a ROI.

Import ROI mask - Clicking on the import ROI mask button will pop up an open file dialog. Users can select an image file to be imported as the specified ROI.

Clear ROI - Clicking on the clear ROI button will clear the highlighted area for the specific ROI.

ROI time course - Clicking on the time plot ROI button will pop up a window showing the plot of average intensity under the bleached area against time.

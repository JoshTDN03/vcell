2. Getting Started
This Chapter covers the following:
  • What are basic configurations to run Virtual Frap
  • How to download and install Virtual Frap

2.1 System Requirements

 In order to run Virtual Frap successfully, your computer must meet the following minimum system requirements.

Windows
 1GHz Intel/AMD CPU
Microsoft Windows 2000/X/Vista
1GB RAM
10GB Free Disk Space

Linux
1GHz CPU
Linux 2.2.* Kernel on i686 processors
1GB RAM
10GB Free Disk Space

Macintosh
1GHz PowerPC G4/Intel CPU
Mac OSX 10.4 and above
1GB RAM
10GB Free Disk Space

NOTE that the current version of Virtual Frap requires Java Runtime Environment (J2SE Runtime Environment 5.0 or later).

2.2 Installation Instructions
  1. Download the Virtual Frap installer according to your operating system from CCAM file server (\\fs3\CCAM\Public\VirtualMicroscopy\installer ). Before you download a file, notice that its byte size is shown in the directory. Once the download has completed, check that you have downloaded the full, uncorrupted software file.

    2.
  2. Execute the InstallVM_”operating system type”.exe which you have downloaded and follow the instructions. Three installation methods are provided, which are typical, custom and complete installations.
  3. The installer will automatically install the software files into directory CCAM/VirtualMicrosopy under user specified parent directory. The default parent directory from installer is C:\Program Files. Note that user specified parent directory should not include white spaces in the path.

    4.
  4. If you have an older version of Virtual Frap already installed, you can install the current version to the same directory and the new version will overwrite the previous.

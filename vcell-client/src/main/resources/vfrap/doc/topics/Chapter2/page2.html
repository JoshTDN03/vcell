System Requirements

 In order to run Virtual Frap successfully, your computer must meet the following minimum system requirements.

Windows
 1GHz Intel/AMD CPU
Microsoft Windows 2000/X/7/8
1GB RAM
10GB Free Disk Space
Monitor resolution 1024 by 768 pixels


Macintosh
 1GHz Intel CPU
Mac OS X 10.x
1GB RAM
10GB Free Disk Space
Monitor resolution 1024 by 768 pixels

Since VFrap performs computations locally, having additional CPU and memory resources will result in noticeable improvement in performance, particularly for large geometries.
